# Build Phases

## Phase 1: Foundation

- Initialize Next.js App Router project
- Configure Tailwind, TypeScript, Prisma, Auth.js, and env validation
- Establish Contentful models and PostgreSQL schema

## Phase 2: Public Experience

- Build CMS data layer and shared block renderer
- Ship all core public routes
- Add metadata, sitemap, robots, preview mode, and revalidation

## Phase 3: Dashboard Operations

- Add auth flow and middleware protection
- Ship submissions, users, roles, activity logs, and profile modules
- Add workflow helper routes for Contentful-managed content

## Phase 4: Editorial Productivity

- Add Contentful sync/index jobs
- Add internal preview validation, QA checklists, and release workflows
- Add media indexing and menu visualization helpers

## Phase 5: Integrations and Scale

- Add CRM integrations, analytics, and background jobs
- Add localization, richer search indexing, and AI-assisted workflow helpers
- Expand export jobs and reporting dashboards
